Freelance - Project Tracker (June?? 2020 - January 2021)
I supported my client in creating Project Tracker, a web application adapted from a legacy software that was created for them several years ago. My senior design team began this project from scratch my last semester of college, and I have continued to develop and support this project as a freelancer since my graduation. Retrievabl (June?? 2020 - May 2020)
I worked independently to create this search engine web application for my Information Retrieval class' term project. It allows the user to search for news articles by using either a default search algorithm (BM25) or a custom search algorithm I developed to filter out news articles with high usages of negative-connotation words.
