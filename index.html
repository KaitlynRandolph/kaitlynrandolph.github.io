Discord Bot (November 2021 - February 2022)
Working from requests from my friends in a Discord server for the mobile game Epic7, I wrote a Discord bot for our server using Python. I was able to utilize libraries in Python to schedule messages to be sent as reminders into the Discord chat related to activities in our game. Also, I was able to take text inputted by users in the chat and use it as search queries on a fansite for the game to quickly provide results to users without having to leave the Discord chat.
Project Tracker (June 2020 - January 2021)
My team supported our client in creating Project Tracker, a web application adapted from a legacy software that was created for them several years ago.
